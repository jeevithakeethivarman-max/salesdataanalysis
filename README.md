# Sales Data Analysis Dashboard

## Overview
Analyzed Superstore sales dataset to identify revenue trends, category performance, and regional profitability.

## Technologies Used
Python, Pandas, Matplotlib, Seaborn

## What I Did
- Performed data cleaning and feature extraction from Order Date
- Analyzed monthly sales trends across years
- Identified top performing product categories
- Compared profit across different regions

## Dataset
Superstore Sales Dataset - 9,994 records

## Key Insights
- Technology category had the highest total sales
- West region was the most profitable
- Sales peaked during Q4 (October - December)
